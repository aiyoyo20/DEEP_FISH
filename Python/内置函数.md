## map()
map() 函数语法：
    map(function, iterable, ...)
参数:
    function -- 函数
    iterable -- 一个或多个序列
返回值
    Python 2.x 返回列表。
    Python 3.x 返回迭代器。

传很多个 iterable，如果有额外的 iterable 参数，并行的从这些参数中取元素，并调用 function。如果一个 iterable 参数比另外的 iterable 参数要短，将以长度最小的为基准。

```
def add(x):
    return x+1

list1 = [1,2,3]
res = map(add, list1)
print(list(res))

# [2, 3, 4]
```
```
from collections.abc import Iterator

def add(x,y,z):
    return x,y,z

list1 = [1,2,3]
list2 = [1,2,3,4]
list3 = [1,2,3,4,5]
res = map(add, list1, list2, list3)
print(list(res))
print(isinstance(res,Iterator))

# [(1, 1, 1), (2, 2, 2), (3, 3, 3)]
# True
```
