19、链式函数调用
在一行代码中调用多个函数。

def add(a, b): 
    return a + b 
 
def subtract(a, b): 
    return a - b 
 
a, b = 5, 10 
 
print((add if b > a else subtract)(a,b)) 

8、查找出现次数最多的元素
使用max方法找出列表中出现次数最多的元素。

def most_frequent(list): 
  return max(set(list), key=list.count) 
 
mylist = [1,1,2,3,4,5,6,6,2,2] 
print("出现次数最多的元素是:", most_frequent(mylist))
